# Agent Handbook

This handbook is the daily reference for the **Systems Architect** role.

## What to use

- **Role contract**: `AGENTS.md`
- **SOPs**: `handbook/sops/`
- **Runbooks**: `handbook/runbooks/`
- **Templates**: `handbook/templates/`
- **References**: `handbook/references/`

## Guidance

- Use `AGENTS.md` as the authoritative role contract.
- General handbook content applies to all roles; role-specific content should live in the same folders with role-scoped titles.
- Use SOPs and runbooks for repeatable workflows.
- Add only the materials required for this role to operate independently.

## General handbook baseline (required for all roles)

- Process-improvement loop SOP
- Efficiency-opportunity issue template
